4. Push 전엔 Pull이 있다

	pull
	- 명령어 : git pull origin master (지정 해놓은 Repository에서 끌어옵니다)


	push
	- 명령어 : git status
	- 명령어 : git add (변경된 파일들을 모두 tracking해죠 명령어)
	- 명령어 : git commit -m 'message' (커밋 메지지)
	- 명령어 : git push origin + master (실제로 git에 push)
